# JavaFX-Bus-Ticket-Booking-System

Bus Ticket Booking System with-\
•	JavaFX\
•	MySQL\
•	Scene Builder

Users Can:\
	Login and Signup\
	Search bus\
	Select bus, see available and book seats, select seats\
	Pay by two payment methods\
	View current bookings


Admin Can:\
	Add new bus\
	View Booking History\
